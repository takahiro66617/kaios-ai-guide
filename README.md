# KAIOS — 社内改善提案システム

社員がアイデアを提出すると、AIが自動で採点・整理してくれる社内ツールです。

> 「社員の小さな改善が、AIによって企業の"知能"へと変換され、組織を進化させ続ける。」

---

## このシステムでできること

- 社員が「こうすればもっと良くなる」という改善アイデアを入力する
- AIが内容を自動で整理し、100点満点で採点する
- 管理者が実行段階を管理できる（提案中 → 実行予定 → 実行済み）
- みんなの改善実績がゲーム感覚で見える（レベルアップ・ミッションなど）
- 未実行のアイデアも「将来の資産」として保存される

---

## 採点のしくみ（100点満点）

採点は **7つの軸** で行われます。大きく3種類に分かれています。

### ① 絶対に使う固定の3軸（合計60点・変更不可）

| 軸の名前 | 点数 | 何を見るか |
|----------|------|-----------|
| **本質性** | 20点 | 表面だけでなく、問題の根っこに触れているか |
| **収益への貢献** | 20点 | 売上アップ・コスト削減など、会社の利益につながるか |
| **実現できるか** | 20点 | 今の業務の中で実際に動かせるか |

これら3軸は**全企業共通**で、時代が変わっても比べられるよう固定されています。

---

### ② 会社の戦略に合わせて選ぶ軸（20点）

以下の候補から **2つ選ぶ**（それぞれ10点）

| 候補 |
|------|
| 顧客価値 |
| 生産性向上 |
| 離職防止 |
| 品質向上 |
| 採用力向上 |
| 標準化 |
| 属人化解消 |

---

### ③ 会社の文化に合わせて選ぶ軸（20点）

以下の候補から **2つ選ぶ**（それぞれ10点）

| 候補 |
|------|
| 誠実 |
| 顧客志向 |
| 主体性 |
| 協働 |
| 学習志向 |

## 実行段階のしくみ

改善アイデアには3つの段階があります。

| 段階 | 意味 | ポイント加算 |
|------|------|------------|
| 📋 **提案中** | まだ動いていない | 採点点数のみ |
| 📅 **実行予定** | これからやる予定 | 採点点数 ＋ 中程度のボーナス |
| ✅ **実行済み** | 実際にやった | 採点点数 ＋ 最大ボーナス（+20点） |

**大事なポイント：**
未実行のアイデアも捨てずに保存します。将来の「人材発掘・経営判断の材料」として活用するためです。

---

## 誰が何をできるか（権限のしくみ）

| 役割 | できること |
|------|-----------|
| **管理者（Admin）** | 選択軸の設定・変更、全社員の評価結果を見る、実行段階の更新 |
| **管理職（Manager）** | 自分の部署の改善案を見る、実行段階の更新 |
| **一般社員（Employee）** | 自分の改善案の点数・AIコメント・実行段階を見る |

## 今後やること（ミーティング 2026年4月30日 決定事項）

### 1. 採点基準を整理する（最優先）

**今の問題：** 採点項目を自由に何でも追加できてしまい、ルールがバラバラ。

**こう直す：**
- 固定3軸（本質性・収益への貢献・実現できるか）をロックして変更できないようにする
- 戦略軸・文化軸はあらかじめ用意したリストから選ぶ方式に変える
- 合計が必ず100点になるよう管理する

---

### 2. 実行ボーナス（+20点）の表示を正しく反映する

**今の問題：** 実行済みにしてもボーナス点が画面に表示されていない。

**こう直す：**
- 「AIの採点（ベース点）＋ 実行ボーナス = 表示点数」という計算を画面に反映する
- ランキングや統計にもボーナス込みの点数を使う

---

### 3. キャラクター診断を追加する（ゲーム要素）

**今の問題：** 自分の画面にはレベルと点数しか表示されていない。もっと楽しくしたい。

**こう直す：**
改善アイデアの傾向から、自分のキャラクタータイプを表示する。

| キャラクター | 特徴 |
|------------|------|
| 🧙 賢者 | 問題の根本を見抜くアイデアが得意（本質性が高い） |
| ⚔️ 戦士 | コスト削減・売上アップに直結するアイデアが得意（収益性が高い） |
| ✨ 魔法使い | 現実的にすぐ動けるアイデアが得意（実現性が高い） |
| 🦸 英雄 | 3つのタイプをバランスよく持っている |

---

## 作業の優先順位

| 順番 | やること | 難易度 |
|------|---------|--------|
| 1 | 採点基準の整理（固定軸のロック・選択方式への変更） | 大きな変更 |

| 3 | キャラクター診断カードの追加 | 中くらいの変更 |
