# KAIOS — 社内改善提案システム

社員がアイデアを提出すると、AIが自動で採点・整理してくれる社内ツールです。

---

## このシステムでできること

- 社員が「こうすればもっと良くなる」という改善アイデアを入力する
- AIが内容を自動で整理し、採点する
- 管理者が承認・却下できる
- みんなの改善実績がゲーム感覚で見える（レベルアップ・ミッションなど）

---

## 今後やること（ミーティング 2026年4月30日 決定事項）

### 1. 採点の基準を整理する

**今の問題:**
採点項目を自由に何でも追加できてしまい、会社によってバラバラになっている。

**こう直す:**

| 種類 | 項目 | 点数の割合 |
|------|------|-----------|
| 絶対に使う（変更不可） | 本質性・収益への貢献・実現できるか | 合計60点分 |
| 会社ごとに選ぶ | リストから2つ選ぶ（例：生産性向上、品質向上 など） | 合計40点分 |

「絶対に使う3項目」は誰でも変えられないようにロックする。
「会社ごとに選ぶ2項目」は、あらかじめ用意した10種類のリストから選ぶ方式にする。

---

### 2. 「権限外」という状態を追加する

**今の問題:**
改善アイデアの状態は「提案中・実行予定・実行済み」の3種類しかない。
でも実際には「やりたいけど自分の権限では動かせない」ケースがある。

**こう直す:**
「権限外」という状態を追加する。

| 状態 | 意味 |
|------|------|
| 提案中 | まだ動いていない |
| 実行予定 | これからやる予定 |
| 実行済み | 実際にやった ✅ |
| 権限外 | やりたいが自分ではできない 🔒 |

また、「実行済み」にすると**最大+20点のボーナス**がつくようにする。
実際に動いた改善が高く評価される仕組み。

---

### 3. キャラクター診断を追加する（ゲーム要素）

**今の問題:**
自分の画面にはレベルと点数しか表示されていない。もっと楽しくしたい。

**こう直す:**
提出した改善アイデアの傾向から、自分のタイプをキャラクターで表示する。

| キャラクター | タイプ | 特徴 |
|------------|--------|------|
| 🧙 賢者 | 本質探求者 | 問題の根本を見抜くアイデアが得意 |
| ⚔️ 戦士 | 収益の戦士 | コスト削減・売上アップに直結するアイデアが得意 |
| ✨ 魔法使い | 実装の魔法使い | 現実的にすぐ動けるアイデアが得意 |
| 🦸 英雄 | 万能の英雄 | 3つのタイプをバランスよく持っている |

---

## 作業の優先順位

1. **採点基準の整理**（他の全てに影響するため最初に対応）
2. **「権限外」の追加**（小さな変更で対応可能）
3. **実行ボーナス+20点の表示**（小さな変更で対応可能）
4. **キャラクター診断カードの追加**（上3つが完成してから対応）
